# Design Profile

## 방향

- 토스, 카카오처럼 정돈되고 신뢰감 있는 툴
- 장식보다 정보 구조 우선
- 지도는 배경, 탐색 UI가 전면

## 컬러

- Primary 500: `#138A9E`
- Primary 600: `#0F6B7A`
- Primary 50: `#DDF4F6`
- Text 900: `#111827`
- Text 600: `#6B7280`
- Background: `#F7FAFB`
- Surface: `#FFFFFF`
- Border: `#E5EEF1`

## UI 원칙

- 대분류는 크게, 소분류는 칩으로
- 카드와 패널은 짧고 단단하게
- 한 화면에서 탐색과 확인이 끝나야 한다
- 화려한 모션보다 빠른 반응이 중요하다

## 카피 톤

- 짧고 직접적
- 설명은 한 줄
- 전문 용어는 그대로 쓰되, 첫 노출에서는 보조 힌트를 붙인다

## 메인 문구

- 앱 이름: `mandi` (임시)
- 홈 메인 카피: `누구나 제작 가능, 소량 제작 업체 총정리`
