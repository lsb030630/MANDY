# Product Profile

## 서비스 한 줄 정의

`mandi`는 소량 제작이 필요한 사람이 작업 업체와 재료상을 빠르게 찾도록 돕는 제작 인프라 앱이다.

## 현재 범위

- 지역: 을지로, 종로
- 대상: 전시, 과제, 시제품 제작을 위해 외부 업체를 찾아야 하는 사람
- 기준: 소량 제작 가능 업체만 다룬다

## 현재 정보 구조

- 첫 진입: `작업 맡기기` / `재료 사기`
- 작업 맡기기 카테고리:
  - 3D프린팅
  - CNC
  - 레이저커팅
  - 용접
  - 벤딩
  - 도장
  - 목공
- 재료 사기 카테고리:
  - 금속
  - 목재
  - 아크릴
  - 천
  - 가죽
  - 기타

## 핵심 UX 원칙

- 지도 앱처럼 보이되, 실제 본체는 업체 데이터베이스다.
- 상세 페이지보다 탐색 화면이 더 중요하다.
- 사용자는 `지도 + 리스트 + 검색` 안에서 대부분의 결정을 끝내야 한다.
- 길고 많은 정보보다, 전화할 수 있는 정보와 짧은 메모가 우선이다.

## 현재 우선순위

1. 위치가 확인된 업체만 노출
2. 검색, 필터, 북마크가 안정적으로 동작
3. 카카오맵 실지도 연결
4. 데이터 계속 추가

## 당장 하지 않는 것

- 회원 시스템
- 공개 리뷰 시스템
- 장문 상세 페이지
- AI 추천 기능의 본격 배포
